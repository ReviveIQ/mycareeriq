# Bryan Greer — Job Search Pipeline Dashboard: Design Ideas

## Idea 1: "Command Center" — Dark Intelligence Dashboard
<response>
<text>
**Design Movement:** Dark-mode data intelligence / Bloomberg Terminal meets modern SaaS
**Core Principles:**
- Information density without clutter — every pixel earns its place
- Monochromatic dark base with electric accent pops (cyan/teal)
- Tabular data as a first-class citizen, not an afterthought
- Sidebar navigation with collapsible sections

**Color Philosophy:** Deep navy (#0a0f1e) background, slate-700 card surfaces, electric cyan (#00d4ff) for primary actions and highlights. Conveys authority and analytical precision.

**Layout Paradigm:** Fixed left sidebar (240px) with pipeline stages, main content area with filterable company table, top metrics bar with KPI cards.

**Signature Elements:**
- Glowing cyan borders on active/hover states
- Monospace font for data cells (JetBrains Mono)
- Subtle grid pattern background texture

**Interaction Philosophy:** Keyboard-first, filter-heavy, data-dense. Every row is a prospect.

**Animation:** Fade-in rows on load, slide-in sidebar panels, smooth filter transitions.

**Typography System:** Syne (headers) + JetBrains Mono (data) + Inter (body)
</text>

## Idea 2: "Territory Map" — Clean Professional Light Dashboard
<response>
<text>
**Design Movement:** Modern B2B SaaS product design — Notion meets Linear
**Core Principles:**
- Clean white canvas with structured information hierarchy
- Warm neutral tones with a single bold accent (deep indigo)
- Cards and tables coexist harmoniously
- Progress tracking as a visual narrative

**Color Philosophy:** Off-white (#fafaf9) background, warm gray cards, deep indigo (#4338ca) as the primary action color. Professional, trustworthy, approachable.

**Layout Paradigm:** Top navigation with pipeline stage filters, full-width content area, collapsible detail drawer on row click.

**Signature Elements:**
- Status badges with semantic color coding (warm amber = outreach, green = active, blue = applied)
- Subtle left-border accent on cards indicating priority
- Thin progress bars showing pipeline stage distribution

**Interaction Philosophy:** Click-to-expand rows, filter chips, sortable columns. Feels like a CRM.

**Animation:** Smooth accordion expand/collapse, subtle hover lift on cards, staggered list entrance.

**Typography System:** DM Sans (headers) + Inter (body) — clean, readable, modern
</text>

## Idea 3: "Sales Floor" — Bold Editorial Dashboard
<response>
<text>
**Design Movement:** Editorial print design meets data visualization — WSJ infographic aesthetic
**Core Principles:**
- Strong typographic hierarchy as the primary design tool
- Warm cream/ivory base with bold charcoal text and gold accents
- Data visualizations as editorial graphics, not just utility charts
- Asymmetric layout with deliberate tension

**Color Philosophy:** Cream (#fdf8f0) background, charcoal (#1a1a2e) text, gold (#d4a017) accents. Evokes confidence, gravitas, and editorial authority.

**Layout Paradigm:** Two-column asymmetric layout — left column (40%) for stats/charts, right column (60%) for the company pipeline table.

**Signature Elements:**
- Large bold numbers for key metrics (30 companies, X applied, X active)
- Horizontal rule dividers with gold accent dots
- Company logos as visual anchors in the table

**Interaction Philosophy:** Scroll-driven narrative, expandable company cards, visual pipeline funnel.

**Animation:** Counter animations for metrics, smooth chart draw-in, editorial page transitions.

**Typography System:** Playfair Display (display headers) + Source Sans Pro (body/data)
</text>

</response>

## Selected Design: Idea 2 — "Territory Map" (Clean Professional Light Dashboard)

This approach best serves Bryan's use case as an active job seeker who needs to work this pipeline daily. The clean, CRM-like interface prioritizes usability and data clarity, with warm neutrals and deep indigo accents that feel professional without being cold or intimidating. The status badge system and filter chips make it easy to track pipeline stage at a glance.
